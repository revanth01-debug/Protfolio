🤖 AI Project

NLP-Based Chatbot

An intelligent chatbot built using Natural Language Processing techniques that understands user queries and responds intelligently using machine learning and NLP.

🎯 Key Features

  • Interactive NLP-based chatbot interface
  • Intent recognition using ML models
  • Predefined knowledge base dataset
  • Real-time response generation
  • Web-based UI using Streamlit
  • Auto-refresh functionality

🛠️ Tech Stack

Python NLTK Streamlit Scikit-learn JSON ML Models

📂 Project Structure

  • app.py - Main Streamlit application
  • intents.json - Training dataset with intents
  • model.pkl - Trained ML model
  • tokenizer.pkl - Text tokenizer
  • requirements.txt - Dependencies
